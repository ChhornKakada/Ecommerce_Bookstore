+ "props" stands for properties. Props are a way to pass data from a parent component to a child component. They allow you to define and pass values from a higher-level component (parent) to a lower-level component (child).

+ `$emit` is a method provided by the Vue instance and Vue components. It is used to emit custom events from a component to its parent or ancestor components.
